# REVERSE

Reverse is another string function, which as the name suggests reverses a given string. 

SELECT REVERSE('Hello world'); 

+------------------------+
| REVERSE('Hello world') |
+------------------------+
| dlrow olleH            |
+------------------------+
1 row in set (0.13 sec)
